position

            position = 겹치다. 이뿐거~ 
            position : static;  = position이 아니게 한다
            position : relative;  = '기준'
            position : absolute;  = 비로소 겹쳐놓는다, 띄운다, 좌표를 가지고 떠오른다. 기준이 없다.
            (떠오르게되면 height를 0으로 만들어 쪼그라든다. > 기준이 body가 된다. (부모고뭐고 없다.)
            > 부모를 있게하고싶으면 relative하거나 absolute에게 absolute를
            먹이면 된다.
            position : fixed;  = absolute와 같은데 기준이 없다.(viewport, 화면 기준이다.)
            화면 밖으로 나가도 스크롤바가 생기지 않는다.

transform

            자기자신을 움직인다. 
            레이아웃을 안깬다.
            값을 줬을 때 자기 자신의 사이즈를 기준으로 반영한다.
            transform을 두줄로 쓰지 않고 한줄에 배치해준다.
            translateX(-50) translateY(-50)를 translate(-50%, -50%)로 줄일 수 있다

오늘의 공식

            1. 중간에 오는 팝업 공식
            position: absolute;
            top: 50%;
            left: 50%;
            transform: translate(-50%, -50%);
