  • 기본 박스 02

    css 선택자에 대한 이해

    css는 계급이 정확해서 부모를 선택할 수 없다. (역행할 수 없고 무조건 아래를 선택)

    뒤에 오는 것은 앞의 내용을 덮어쓴다.

    important > inline style의 속성 > id > class

    1. * : (body까지 포함하는) 모든 선택자
    2. 공간 : tag 선택자와 후손 선택자
    3. ~ : (나를 제외한) 형제선택자. 붙여쓰기!
    4. + : (내 아래의) 인접형제선택자. 붙여쓰기!
    5. [class=""], [class*=""] : attribute 선택자
    6. , : 그리고 (따옴표 안에 각각의 위치를 정확하게 적는다.)
    7. nth-child(-) : -번째자식 / nth-of-type(-) : 같은 태그중에 -번
