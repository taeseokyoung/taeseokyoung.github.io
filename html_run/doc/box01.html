box의 기본성질 (display:block)

  • 투명하다.
  • width: auto / height: auto
  • Y축으로 쌓인다.
  • block 옆에 block이 오지 못하게 한다.
  1. 태그는 크게 inline 과 block 으로 나뉜다.
  2. 글자 그림 등 내용에 관여하는 태그 span strong em icon 은 인라인 속성을 가지고
    나머지는 block 속성을 가진다.
  3. block 속성은 width height margin padding 값을 가진다.
  4. inline 속성은 좌우 margin padding 만을 가진다. (줄 수 있다.)
    inline의 상하 padding의 경우 background를 포함하여 padding이 적용된 것처럼 보인다.
* input태그는 다양한 타입이 있다. focus 와 change(event)

CSS는 레이아웃을 만들어준다.

  • 인라인 스타일이 외부/내부 스타일보다 우선한다. (계급이 높다.)
    js의 적용범위는 인라인 스타일이므로 외부/내부 스타일보다 우선한다.
  • 외/내부 스타일 시트 계급은 뒤에 온 것이 앞의 것을 덮어쓴다.
  • 밖에(head 안에) 만드는 이유? 모듈화 = 컴포넌트(리액트)
    가끔 밖에 만든 모듈을 건드리기 어려울 때 임시방편으로 바로 바꿔줄 수 있다
  • style name role : attribute, property / props(리액트).
